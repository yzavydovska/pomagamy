# 08 — Backend w repozytorium: Firestore i Storage

- **`mobile/firestore.rules`**: dostępy do kolekcji użytkowników, ogłoszeń, zgłoszeń, skarg, organizacji, powiadomień wbudowanych oraz ról moderatora nad zwykłymi użytkownikami.
- **`mobile/storage.rules`**: ścieżki m.in. **avatary**, dokumenty **`orgDocs`** (statut przy rejestracji organizacji), zgodnie z polityką autoryzacji po `uid`.
